Dev Tools
by Hazer
📦 Description

This mod for ICARUS unlocks and enhances various developer tools and debug features. It provides advanced users and modders with the necessary utilities to inspect game data, test mechanics, and streamline the creation of new content within the game environment.

Perfect for players who:

want to experiment with hidden engine features and debug commands

are looking to develop or test their own mods for ICARUS

need advanced diagnostic tools to analyze gameplay behavior

⚙️ Features

🔼 Access Developer Console: Unlocks advanced debug and utility commands

🛠️ Inspection Utilities: Tools to view and analyze real-time game data

⚡ Creation Workflow: Features designed to speed up testing and mod development

🔧 Easy to install: Simple setup for immediate access to dev features

📥 Installation

Download the mod files

Navigate to your ICARUS installation folder:
Icarus\Content\Paks\

Place the mod file there (e.g. .pak file)

Launch the game

❌ Uninstallation

Remove the mod file from the Paks folder

Restart the game

⚠️ Notes

Use with caution: Developer tools can significantly alter or break your game state if used incorrectly

May be incompatible with other mods that modify the game's core UI or console access

It is highly recommended to back up your save files before using these tools

🔄 Compatibility

Tested with the current version of ICARUS (at time of release)

Compatible with most standard mods, but may conflict with other "Cheat" or "Trainer" style mods
