Range Extended
by Hazer
📦 Description

V. 1.2. 

Range extended, plus OuterRange for the Large Heater

This mod for ICARUS significantly extends the effective range of temperature-altering objects. It allows devices like Heaters, Fireplaces, Coolers, and Firepits to cover a much larger area, ensuring your base stays comfortable without needing to place an excessive number of units.

Perfect for players who:

build large bases and want full temperature coverage

want to reduce the clutter of having multiple heaters or coolers in one room

prefer a more realistic and powerful area-of-effect for thermal devices

⚙️ Features

🔼 Extended Thermal Range: Increased AOE for Heaters, Fireplaces, Coolers, and Firepits

🏠 Base-Wide Comfort: Heat or cool larger rooms with a single device

⚡ No Performance Impact: Optimized for smooth gameplay

🔧 Easy to install: Simple drag-and-drop setup

📥 Installation

Download the mod files

Navigate to your ICARUS installation folder:
Icarus\Content\Paks\

Place the mod file there (e.g. .pak file)

Launch the game

❌ Uninstallation

Remove the mod file from the Paks folder

Restart the game

⚠️ Notes

May be incompatible with other mods that modify deployable thermal values

Works best if all multiplayer participants have the mod installed

Use mods at your own risk, especially after game updates

🔄 Compatibility

Tested with the current version of ICARUS (at time of release)

Should be compatible with most QoL mods
