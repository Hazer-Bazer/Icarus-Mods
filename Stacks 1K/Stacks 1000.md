Stacks 1000
by Hazer

📦 Description

This mod for ICARUS increases the maximum stack size for a vast majority of items to 1000. This is the ultimate inventory management solution, allowing you to carry and store massive quantities of resources with ease.

Perfect for players who:

want to practically ignore stack limits

are tired of managing dozens of storage chests

want the ultimate "Quality of Life" improvement for resource hoarding

⚙️ Features

🔼 Ultimate Stack Increase: Most items now stack up to 1000 units

🧳 Extreme Optimization: Maximum storage efficiency for your base

⚡ No Performance Impact: Optimized for smooth gameplay

🔧 Easy to install: Simple installation process


📥 Installation

Download the mod files

Navigate to your ICARUS installation folder: Icarus\Content\Paks\

Place the mod file there (e.g. .pak file)

Launch the game

❌ Uninstallation

Remove the mod file from the Paks folder

Restart the game

⚠️ Notes

May be incompatible with other mods that modify item stack sizes

Works best if all multiplayer participants have the mod installed

Use mods at your own risk, especially after game updates

🔄 Compatibility

Tested with the current version of ICARUS (at time of release)

Should be compatible with most QoL mods
