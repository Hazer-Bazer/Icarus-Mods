# Hazers Fillable Increased

## 📦 Description

**Hazers Fillable Increased** is a mod for **ICARUS** that increases the **capacity of fillable items**, allowing you to store and carry more resources in containers such as water or other fillable objects.

Created by **Hazer / Shelly3000**, this mod is designed to reduce micromanagement and improve overall efficiency by minimizing the need to constantly refill or carry multiple containers.

Perfect for players who:

* want larger capacities for fillable items
* are tired of frequent refilling
* prefer smoother resource management
* focus on exploration or long trips

---

## ⚙️ Features

* 💧 Increased capacity for fillable items
* 📦 Carry more resources with fewer containers
* 🧭 Ideal for long exploration runs
* ⚡ Reduces micromanagement and interruptions
* 🚀 Lightweight and balanced quality-of-life improvement

---

## 📥 Installation

1. Download the mod files
2. Navigate to your ICARUS installation folder:

   ```
   Icarus\Content\Paks\
   ```
3. Place the mod file there (e.g. `.pak` file)
4. Launch the game

---

## ❌ Uninstallation

1. Remove the mod file from the `Paks` folder
2. Restart the game

---

## ⚠️ Notes

* May conflict with other mods that modify item capacities or resource systems
* In multiplayer, it is recommended that all players use the same mod setup
* Use mods at your own risk, especially after game updates

---

## 🔄 Compatibility

* Tested with the current version of ICARUS (at time of release)
* Compatible with most Quality-of-Life mods unless they alter fillable item behavior
