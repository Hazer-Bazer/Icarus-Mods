# Hazers More Start Backpack Slots

## 📦 Description

**Hazers More Start Backpack Slots** is a mod for **ICARUS** that increases the **number of available inventory slots at the start of the game**, giving players more space right from the beginning.

Created by **Hazer / Shelly3000**, this mod reduces early-game inventory limitations and allows for a smoother and less restrictive start, especially useful for exploration and resource gathering.

Perfect for players who:

* want more inventory space from the start
* dislike early-game inventory restrictions
* prefer smoother exploration and looting
* play solo or with limited storage options early on

---

## ⚙️ Features

* 🎒 Increased starting backpack slots
* 📦 More inventory space from the beginning
* 🧭 Better early-game exploration experience
* ⚡ Reduces inventory micromanagement
* 🚀 Lightweight and balanced improvement

---

## 📥 Installation

1. Download the mod files
2. Navigate to your ICARUS installation folder:

   ```
   Icarus\Content\Paks\
   ```
3. Place the mod file there (e.g. `.pak` file)
4. Launch the game

---

## ❌ Uninstallation

1. Remove the mod file from the `Paks` folder
2. Restart the game

---

## ⚠️ Notes

* May conflict with other mods that modify inventory or backpack size
* In multiplayer, it is recommended that all players use the same mod setup
* Use mods at your own risk, especially after game updates

---

## 🔄 Compatibility

* Tested with the current version of ICARUS (at time of release)
* Compatible with most Quality-of-Life mods unless they alter inventory systems
