Stacks 250
by Hazer
📦 Description

This mod for ICARUS increases the maximum stack size for a vast majority of items to 250. It is designed to declutter your inventory and storage containers, making resource management much more efficient.

Perfect for players who:

are tired of overflowing chests and limited inventory space

want to carry large amounts of resources during long mining trips

prefer a cleaner, more organized storage system

⚙️ Features

🔼 Massive Stack Increase: Most items now stack up to 250 units

🧳 Inventory Optimization: Carry more materials in fewer slots

⚡ No Performance Impact: Optimized for smooth gameplay

🔧 Easy to install: Simple installation process

📥 Installation

Download the mod files

Navigate to your ICARUS installation folder:
Icarus\Content\Paks\

Place the mod file there (e.g. .pak file)

Launch the game

❌ Uninstallation

Remove the mod file from the Paks folder

Restart the game

⚠️ Notes

May be incompatible with other mods that modify item stack sizes

Works best if all multiplayer participants have the mod installed

Use mods at your own risk, especially after game updates

🔄 Compatibility

Tested with the current version of ICARUS (at time of release)

Should be compatible with most QoL mods
