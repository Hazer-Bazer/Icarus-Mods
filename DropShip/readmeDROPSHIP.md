Increase DropShip Slots (15 → 30) by Hazer


📦 Description

This mod for ICARUS increases the number of available DropShip slots from 15 to 30, allowing you to bring significantly more gear and items into your missions without being limited by the default cap.

Perfect for players who:

want to use larger loadouts

prefer fewer back-and-forth trips between missions

need more flexibility when playing with mods or in multiplayer




⚙️ Features


🔼 Increases DropShip slots from 15 → 30


🧳 More space for equipment & resources


⚡ No performance Impact


🔧 Easy to install and remove





📥 Installation


Download the mod files

Navigate to your ICARUS installation folder:

Icarus\Content\Paks\

Place the mod file there (e.g. .pak file)

Launch the game




❌ Uninstallation


Remove the mod file from the Paks folder

Restart the game



⚠️ Notes


May be incompatible with other mods that modify DropShip or inventory values

Works best if all multiplayer participants have the mod installed

Use mods at your own risk, especially after game updates




🔄 Compatibility


Tested with the current version of ICARUS (at time of release)

Should be compatible with most QoL mods
