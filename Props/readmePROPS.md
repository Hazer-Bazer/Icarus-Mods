# Hazers Props



## 📦 Description


**Hazers Props** is a mod for **ICARUS** that unlocks and makes use of **already existing in-game props** that were previously unavailable to players.

These props could not be crafted before — until now. With this mod, they are accessible via the **Fabricator**, allowing you to fully utilize hidden or unused assets and expand your building possibilities.

Created by **Hazer / Shelly3000**, this mod enhances your creative freedom by bringing previously inaccessible content into normal gameplay.

Perfect for players who:

* enjoy base building and decoration
* want access to hidden or unused in-game props
* like creating immersive and highly detailed outposts

---

## ⚙️ Features

* 🧱 Unlocks **existing but previously uncraftable props**
* 🏭 Props are now craftable in the **Fabricator**
* 🎨 Expands building and decoration options
* 🏕️ Enhances immersion with more environmental detail
* ⚡ Lightweight and performance-friendly

---

## 📥 Installation

1. Download the mod files
2. Navigate to your ICARUS installation folder:

   ```
   Icarus\Content\Paks\
   ```
3. Place the mod file there (e.g. `.pak` file)
4. Launch the game

---

## ❌ Uninstallation

1. Remove the mod file from the `Paks` folder
2. Restart the game

---

## ⚠️ Notes

* May conflict with other mods that modify props or crafting recipes
* In multiplayer, it is recommended that all players install the mod
* Use mods at your own risk, especially after game updates
