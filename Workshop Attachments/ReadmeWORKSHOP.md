

Hazers Workshop Attachments
by Hazer


📦 Description


This mod for ICARUS revolutionizes your gear customization by adding upgrade slots to all weapons, armors, and tools. This allows you to further enhance and specialize your equipment at the Alteration Bench, giving you the edge needed for the toughest prospects.



Perfect for players who:

want to push their high-tier gear to its absolute limit

enjoy deep customization and RPG-style equipment upgrades

want more utility and power from their existing workshop tools and armor



⚙️ Features


🔼 Universal Upgrade Slots: Added slots to all weapons, armors, and tools

🛠️ Alteration Bench Ready: Fully compatible with the existing alteration system

⚡ Enhanced Gear Progression: Take your favorite equipment to the next level

🔧 Easy to install: Seamless integration into your current save



📥 Installation

Download the mod files

Navigate to your ICARUS installation folder:
Icarus\Content\Paks\

Place the mod file there (e.g. .pak file)

Launch the game




❌ Uninstallation

Remove the mod file from the Paks folder

Restart the game



⚠️ Notes

May be incompatible with other mods that modify base item attributes or slot counts

Already existing items might need to be re-crafted or updated to show new slots

Use mods at your own risk, especially after game updates



🔄 Compatibility

Tested with the current version of ICARUS (at time of release)

Should be compatible with most QoL mods
