Queue Increased
by Hazer
📦 Description

This mod for ICARUS expands the crafting queue for a wide variety of workbenches. It allows you to queue significantly more crafting orders at once, eliminating the need to constantly monitor your production.

Perfect for players who:

need to process large amounts of resources

want to use their time more efficiently while workbenches are active

prefer automated production for complex building projects

⚙️ Features

🔼 Increased Queue Slots: Significantly more slots for crafting orders

🛠️ Broad Support: Works with most common workbenches

⚡ No Performance Impact: Runs smoothly without affecting game performance

🔧 Easy to install: Quick setup with no complications

📥 Installation

Download the mod files

Navigate to your ICARUS installation folder:
Icarus\Content\Paks\

Place the mod file there (e.g. .pak file)

Launch the game

❌ Uninstallation

Remove the mod file from the Paks folder

Restart the game

⚠️ Notes

May be incompatible with other mods that modify crafting or workbench values

Works best if all multiplayer participants have the mod installed

Use mods at your own risk, especially after game updates

🔄 Compatibility

Tested with the current version of ICARUS (at time of release)

Should be compatible with most QoL mods
