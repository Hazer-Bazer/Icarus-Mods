Stacks 500
by Hazer

📦 Description

This mod for ICARUS increases the maximum stack size for a vast majority of items to 500. It provides a significant boost to your carrying capacity and drastically reduces the number of storage containers needed in your base.

Perfect for players who:

manage large-scale industrial bases

want to carry twice as many resources as the 250-stack version

prefer a highly organized and compact inventory

⚙️ Features

🔼 Massive Stack Increase: Most items now stack up to 500 units

🧳 Inventory Optimization: Carry massive amounts of materials in fewer slots

⚡ No Performance Impact: Optimized for smooth gameplay

🔧 Easy to install: Simple installation process

📥 Installation

Download the mod files

Navigate to your ICARUS installation folder: Icarus\Content\Paks\

Place the mod file there (e.g. .pak file)

Launch the game

❌ Uninstallation

Remove the mod file from the Paks folder

Restart the game

⚠️ Notes

May be incompatible with other mods that modify item stack sizes

Works best if all multiplayer participants have the mod installed

Use mods at your own risk, especially after game updates

🔄 Compatibility

Tested with the current version of ICARUS (at time of release)

Should be compatible with most QoL mods
