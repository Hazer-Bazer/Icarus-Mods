Hazers ALL IN ONE
📦 Description

Hazers ALL IN ONE is a comprehensive mod for ICARUS that combines multiple quality-of-life improvements from other mods by Hazer / Shelly3000 into a single package.

This all-in-one solution enhances various aspects of the game, including energy production, inventory management, loadout capacity, and more, providing a smoother and less restrictive gameplay experience.

Perfect for players who:

want multiple QoL improvements in one mod
prefer a streamlined and convenient setup
enjoy less grind and more efficiency
don’t want to install and manage multiple individual mods
⚙️ Features
⚡ Increased energy production
🔋 Reduced energy limitations and improved efficiency
🎒 Increased backpack slots
🚀 Expanded DropShip loadout capacity
💧 Increased capacity for fillable items
📦 Improved inventory and storage management
🧩 Combines multiple mods into one easy installation
📥 Installation
Download the mod files

Navigate to your ICARUS installation folder:

Icarus\Content\Paks\
Place the mod file there (e.g. .pak file)
Launch the game
❌ Uninstallation
Remove the mod file from the Paks folder
Restart the game
⚠️ Notes
Do not use together with the individual mods included in this package (may cause conflicts)
May conflict with other mods that modify the same systems (energy, inventory, loadout, etc.)
In multiplayer, it is recommended that all players use the same mod setup
Use mods at your own risk, especially after game updates
🔄 Compatibility
Tested with the current version of ICARUS
Compatible with most mods unless they overlap with included features
