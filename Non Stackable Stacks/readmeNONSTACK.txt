# Hazers Non-Stackable Stacks

## 📦 Description

**Hazers Non-Stackable Stacks** is a mod for **ICARUS** that makes previously **non-stackable items stackable**, allowing you to store multiple units of those items in a single inventory slot.

Created by **Hazer / Shelly3000**, this mod significantly improves inventory management and reduces clutter caused by items that normally cannot stack.

Perfect for players who:

* want cleaner and more efficient inventories
* are frustrated by non-stackable items
* prefer less storage micromanagement
* enjoy a smoother gameplay experience

---

## ⚙️ Features

* 📦 Enables stacking for previously non-stackable items
* 🧳 Reduces inventory clutter significantly
* 🏗️ Improves storage efficiency
* ⚡ Less micromanagement required
* 🚀 Strong quality-of-life improvement

---

## 📥 Installation

1. Download the mod files
2. Navigate to your ICARUS installation folder:

   ```
   Icarus\Content\Paks\
   ```
3. Place the mod file there (e.g. `.pak` file)
4. Launch the game

---

## ❌ Uninstallation

1. Remove the mod file from the `Paks` folder
2. Restart the game

---

## ⚠️ Notes

* May conflict with other mods that modify stack sizes or item behavior
* In multiplayer, it is recommended that all players use the same mod setup
* Use mods at your own risk, especially after updates
