Stacks 750

by Hazer



📦 Description



This mod for ICARUS increases the maximum stack size for a vast majority of items to 750. This version is tailored for heavy-duty resource gathering and extensive building projects where standard stack limits are too restrictive.



Perfect for players who:



focus on massive architectural builds



spend long sessions mining and gathering without returning to base



want a superior balance between inventory freedom and game progression



⚙️ Features



🔼 Superior Stack Increase: Most items now stack up to 750 units



🧳 Advanced Optimization: Significantly reduces chest clutter



⚡ No Performance Impact: Optimized for smooth gameplay



🔧 Easy to install: Simple installation process



**📥 Installation**



**Download the mod files**



**Navigate to your ICARUS installation folder: Icarus\\Content\\Paks\\**



**Place the mod file there (e.g. .pak file)**



**Launch the game**



**❌ Uninstallation**



**Remove the mod file from the Paks folder**



**Restart the game**



**⚠️ Notes**



**May be incompatible with other mods that modify item stack sizes**



**Works best if all multiplayer participants have the mod installed**



**Use mods at your own risk, especially after game updates**



**🔄 Compatibility**



**Tested with the current version of ICARUS (at time of release)**
